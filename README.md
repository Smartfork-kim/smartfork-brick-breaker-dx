# SMARTFORK Brick Breaker DX

HTML5 arcade brick breaker game by SMARTFORK.

## Play

- Move: `← →`, `A / D`, or mouse
- Start/Pause: `Space`
- Laser: `Z`
- Charge Skill: `X`
- Release catch ball: `C`

## Features

- Breakout-style score and speed pressure
- Super Breakout-style multi-ball
- Arkanoid-style power capsules: wide paddle, multi-ball, laser, catch, pierce, slow, shield, 1UP
- Multiple brick types: normal, multi-hit, bomb, bonus, metal/indestructible
- Enemy drones, combo scoring, skill gauge, neon particle effects
- No server required; runs as a single static HTML file

## Privacy

This game stores only the best score in the browser localStorage. No server-side tracking is included by default.
